## 📝 使用示例

### 基本使用流程

```python
# 1. 加载YAML配置
from easyxt_backtest.config import load_strategy_config
config = load_strategy_config('my_strategy.yaml')

# 2. 创建策略
from easyxt_backtest.strategies import ConfigDrivenStrategy
strategy = ConfigDrivenStrategy(config, data_manager)

# 3. 运行回测
from easyxt_backtest import BacktestEngine
engine = BacktestEngine(data_manager=data_manager)
result = engine.run_backtest(strategy, start_date, end_date)

# 4. 查看结果
result.print_summary()
```

### YAML配置示例

```yaml
strategy:
  name: "小市值策略"
  description: "选择市值最小的10只股票"

backtest:
  start_date: "20200101"
  end_date: "20231231"
  initial_cash: 1000000

universe:
  type: "index"
  index_code: "399101.SZ"

exclude_filters:
  - name: "排除ST"
    type: "stock_status"
    condition: "not_in"
    values: ["ST", "*ST"]

scoring_factors:
  - name: "市值因子"
    factor_type: "fundamental"
    field: "market_cap"
    direction: -1
    weight: 1.0
    normalize: true

portfolio:
  select_method: "top_n"
  top_n: 10
  weight_method: "equal"

rebalance:
  frequency: "monthly"
  rebalance_day: 1
```
