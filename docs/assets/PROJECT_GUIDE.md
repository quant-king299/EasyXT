## 🚀 三步上手，超级简单！

### 第一步：看看你的环境准备好了没⚙️

打开命令行（Windows按 Win+R，输入 `cmd`），输入：

```bash
python tools/check_env.py
```

这个工具会帮你检查：
- ✅ Python版本对不对
- ✅ xtquant装好了没
- ✅ EasyXT能不能用
- ✅ DuckDB数据库有没有

**如果看到绿色的✓，恭喜你，环境OK了！👍**

---

### 第二步：跟着教程，做个小练习📝

我们准备了超详细的教程，从最简单的开始：

```python
# 教程1：获取股票数据（超简单！）
from easy_xt import get_api

api = get_api()
data = api.get_full_history(
    ['000001.SZ'],  # 平安银行
    period='1d',     # 日线数据
    start_time='20240101'
)

print(data.head())
# 就这么简单！🎉
```

**学习路径**：
1. 📖 `学习实例/01_基础入门.py` - 5分钟搞定数据获取
2. 📖 `学习实例/02_交易基础.py` - 学会买卖操作
3. 📖 `学习实例/04_策略回测.py` - 写你的第一个策略
4. 📖 `学习实例/09_多数据源统一接口.py` - 玩转多个数据源

每个教程都有详细注释，跟着敲一遍就会了！💪
