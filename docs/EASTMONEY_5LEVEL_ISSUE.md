# 东方财富五档盘口数据问题说明

> **问题报告时间**：2026-03-12
> **状态**：✅ 已修复

---

## 📋 问题描述

有用户反馈：**东方财富EastmoneyDataProvider获取五档数据有问题，现有的bid1、ask1及对应的bid1_vol和ask1_vol也有问题**

---

## ✅ 修复方案

### 方案：使用腾讯API补充五档数据

**实现方式：**
- 东方财富API继续提供基础行情、K线、热度、板块数据
- 腾讯API补充五档盘口数据
- 两个数据源整合，提供完整的市场数据

**代码实现：**
```python
def get_realtime_quotes(self, codes: List[str]):
    # 1. 使用东方财富API获取基础行情
    quotes = self._get_eastmoney_quotes(codes)

    # 2. 使用腾讯API补充五档盘口
    quotes = self._enrich_with_tencent_5level(quotes)

    return quotes
```

**腾讯API字段映射：**
```python
# 买一到买五: 字段 9, 11, 13, 15, 17 (价格)
#           字段 10, 12, 14, 16, 18 (量/手)
# 卖一到卖五: 字段 19, 21, 23, 25, 27 (价格)
#           字段 20, 22, 24, 26, 28 (量/手)

# 注意：腾讯API的成交量单位是"手"，需要乘以100转换为"股"
bid1_vol = int(fields[10]) * 100
```

### 数据流程

```
┌─────────────────────────────────────────────────────────┐
│            EastmoneyDataProvider 数据流程                 │
├─────────────────────────────────────────────────────────┤
│                                                          │

│         │                                                │
│         ▼                                                │
│  ┌──────────────────────┐                               │
│  │ 东方财富API           │                               │
│  │ - 基础行情            │                               │
│  │ - K线数据            │                               │
│  │ - 热度排行           │                               │
│  └──────────┬───────────┘                               │
│             │                                             │

│  ┌──────────────────────┐                               │
│  │ 腾讯API              │                               │
│  │ - 五档盘口数据        │                               │

```
