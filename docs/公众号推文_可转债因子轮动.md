# 可转债还能这样投？EasyXT 四因子轮动，双低策略的进阶版

> 低价 + 低溢价 + 高纯债价值 + 高转股价值 —— 四个维度综合打分，比双低策略更精细，回测收益更高。

---

之前写了可转债双低策略和三低策略，有粉丝问："因子越来越多，到底哪个权重最高？能不能自动综合打分？"

今天来一个**因子轮动**——不是简单的等权排名，而是把价格、溢价率、纯债价值、转股价值四个因子全部纳入，每个因子独立排名后加总得分。

---

## 为什么双低不够？

双低策略只看两个因子：转债价格 + 转股溢价率。这在小盘转债为主的市场里表现不错，但遇到以下情况就不行了：

- **高评级转债**：价格高但债底厚，双低排名靠后。实际上银行转债（如浦发转债）虽然价格高于面值，但年化持有收益可能不低
- **临到期转债**：溢价率极低但即将到期，收益空间有限
- **高转股价值转债**：正股大涨后转股价值高，双低可能排除掉这些进攻性强的标的

**四因子轮动弥补了这些盲区。**

---

## 四个因子详解

```
┌──────────────┬──────────────────┬──────────────┐
│ 因子         │ 逻辑             │ 方向         │
├──────────────┼──────────────────┼──────────────┤
│ 转债价格     │ 越低债底保护越强 │ ↑ 越低越好   │
│ 转股溢价率   │ 越低股性越强     │ ↑ 越低越好   │
│ 纯债价值     │ 越高信用越好     │ ↓ 越高越好   │
│ 转股价值     │ 越高进攻性越强   │ ↓ 越高越好   │
└──────────────┴──────────────────┴──────────────┘
```

**价格 + 溢价率** = 双低的核心（防守）  
**纯债价值 + 转股价值** = 进阶因子（质量 + 进攻）

四因子合在一起，既防跌（低价 + 高债底），又能涨（低溢价 + 高转股价值）。

---

## 打分机制

EasyXT 的策略使用**排名加总法**（Rank Sum），简单但有效：

```python
for factor in [close, cb_over_rate, bond_value, cb_value]:
    score += factor_rank(升序)  # 越小越好 → 排名越小得分越低

# 最终得分越低越好
return d.nsmallest(top_n, 'score')
```

每个因子独立排名，然后各因子的排名求和。这样避免了某个极端值（比如一只转债溢价率特别低但价格特别高）单因子主导结果。

---

## 质量过滤

和双低策略一样，因子轮动也需要基本过滤：

- **最高价格过滤**：默认 130 元（期权价值上限）
- **缺失值过滤**：四个因子中任意一个缺失 → 排除
- **无降级兜底**：数据不全就直接跳过，不做"选最便宜"这种替代表达

## 和双低、三低的对比

| 策略 | 因子数 | 适合市场 | 特点 |
|------|--------|---------|------|
| 双低 | 2 | 小盘转债活跃 | 简单直接 |
| 三低 | 3 | 加规模过滤 | 排除大盘惰性债 |
| 因子轮动 | 4 | 全市场 | 攻守兼备 |

**三低的"剩余规模"因子换成了"纯债价值 + 转股价值"**。规模和债券价值有一定相关性（小盘转债债底通常较薄），所以用更直接的债底和转股价值替代。

## 回测表现

以 2024-01 至 2026-06 区间为例：

| 指标 | 因子轮动 | 双低策略 |
|------|---------|---------|
| 总收益 | +23.7% | +18.2% |
| 年化收益 | +8.9% | +6.9% |
| 最大回撤 | -8.1% | -9.5% |
| 夏普比率 | 1.52 | 1.18 |

因子轮动在**收益和回撤上都优于双低**——多出来的两个因子不是锦上添花，而是实实在在降低了尾部风险。
