# 涨停板选股策略：专抓接近涨停但还没封板的强势股

> 涨停板附近往往是主力资金博弈最激烈的区域。本文介绍 EasyXT 的涨停板策略——专选涨幅 9%-10% 的准涨停股，配合流动性过滤。

---

A 股市场的涨停板制度（±10%）造就了一个独特现象：**涨停板附近的股票，第二天大概率高开**。

统计数据显示：当天涨停的股票，次日开盘平均涨幅 +2.5%，溢价率高达 60%+。这就是"涨停板溢价"——资金追涨停板，但涨停封死后买不进去，只能第二天竞价抢。

EasyXT 的涨停板策略专抓这种机会：**选那些接近涨停但还没有封板的股票**。

---

## 策略逻辑

```
1. 全市场扫描
   筛选当日涨幅 9%-10% 的股票

2. 流动性过滤
   排除日成交额 < 100 万的冷门票

3. 按涨幅排序
   取涨幅最大的前 N 只

4. 次日开盘卖出
   吃涨停板溢价
```

---

## 关键设计：为什么是 9%-10% 而不是 10%？

涨停板策略最微妙的地方就在这里：

- **涨幅 = 10%** → 已经封死涨停，买不进去了
- **涨幅 9%-9.5%** → 在涨停板附近试探，还没封死
- **涨幅 < 9%** → 不够强，第二天溢价不明显

策略的目标是涨幅 **9%-10%** 的股票——离涨停一步之遥，主力还在吃货，散户还有机会上车。

---

## 流动性过滤的重要性

涨停板附近成交量巨大，如果没有流动性过滤，很容易选到：

- 单日成交仅几十万的"僵尸涨停"（对倒拉出来的）
- 次日直接低开闷杀

所以策略硬性要求：**日成交额 ≥ 100 万**，排除所有流动性不足的标的。

## 策略实现

核心选股逻辑（`backtest_wrappers.py`）：

```python
def limit_up_daily(df: pd.DataFrame, **params) -> list:
    """涨停板策略：选涨幅 9%-10% 的准涨停股"""
    buy_n = params.get('buy_n', 5)

    if df.empty:
        return []

    # 兼容不同数据源的列名
    pct_col = None
    if 'pct_change' in df.columns:
        pct_col = 'pct_change'
    elif 'pct_chg' in df.columns:
        df['pct_change'] = df['pct_chg']
        pct_col = 'pct_change'
    else:
        return []

    # 涨幅 9%-10%
    d = df.dropna(subset=[pct_col])
    candidates = d[(d[pct_col] >= 9.0) & (d[pct_col] <= 10.0)]

    # 如果当天没有准涨停股，选涨幅最高的作为候选
    if candidates.empty:
        candidates = d.nlargest(buy_n * 3, pct_col)

    # 排除流动性差的
    if 'amount' in candidates.columns:
        candidates = candidates[candidates['amount'] >= 1e6]

    return candidates.nlargest(buy_n, pct_col)['ts_code'].tolist()
```

## 风险提示 ⚠️

涨停板策略是所有策略中**风险最高**的之一：

1. **高波动**：涨停板股的日内波动可能在 ±5% 以上
2. **流动性风险**：涨停股第二天可能直接低开
3. **市场环境敏感**：熊市中涨停溢价大幅缩水
4. **不适合大资金**：涨停股容量有限

**建议**：只用小仓位试水（不超过总仓位的 20%），严格设置止损（-5%）。

---

## 适合谁？

- **短线交易者**：持有周期 1-3 天
- **打板选手**：做涨停板溢价的短线套利
- **有一定风控经验的投资者**：会设止损

**不适合**：长线投资者、追求稳定收益的、新手刚入市。
