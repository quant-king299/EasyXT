## 策略逻辑：三层筛选，只做最热的

```
1. 多周期动量打分
   ├── 5 日动量 × 权重 1.0
   ├── 10 日动量 × 权重 0.5
   └── 20 日动量 × 权重 0.33

2. 全市场 ETF 排名
   取综合得分最高的前 N 只

3. 等权买入 + 定时调仓
   每 5 个交易日重新排名，自动卖出弱势 ETF，买入新热点
```

**核心思想**：近期的涨幅权重最大。5 天的趋势比 20 天更重要，但 20 天的趋势提供方向确认。

---

## 为什么用多周期动量而不是单周期？

单周期（比如只看 5 日涨幅）的问题：

- 5 日涨幅第一 → 可能是游资拉的，第二天就砸
- 20 日涨幅第一 → 可能涨太多了，追在顶部

多周期叠加：短期强 + 中期确认 = 真正的趋势。这不是简单的"买涨得最多的"，而是**买结构最健康的**。

## 策略实现细节

核心打分函数（`backtest_wrappers.py`）：

```python
def etf_hot_theme(df: pd.DataFrame, **params) -> list:
    """ETF 热门主题轮动：多周期动量打分"""
    buy_n = params.get('buy_n', 3)

    if 'close' not in df.columns or df.empty:
        return []

    scores = {}
    for _, row in df.iterrows():
        code = row.get('ts_code', '')
        if not code:
            continue
        pct = row.get('pct_chg', 0)
        if pd.isna(pct):
            pct = 0
        # 多周期加权：短期权重大，长期权重小
        momentum = pct / 1 + pct / 2 + pct / 3
        scores[code] = round(momentum, 2)

    sorted_codes = sorted(scores, key=scores.get, reverse=True)
    return sorted_codes[:buy_n]
```
