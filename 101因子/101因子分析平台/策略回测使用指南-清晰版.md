## 📥 首次使用必看：下载市值数据

### 为什么要下载数据？
- 小市值策略需要按市值选股
- 本地数据速度快（秒级响应）
- 在线API速度慢且不稳定

### 下载方法（超简单）：

#### 方法1：从101因子平台的主GUI下载（推荐）⭐

1. **启动主GUI**
   ```bash
   cd "C:\Users\Administrator\Desktop\miniqmt扩展"
   python run_gui.py
   ```

2. **找到下载页面**
   - 点击 **"📥 Tushare数据下载"** 标签
   - 选择 **"💰 市值数据"** 子标签

3. **一键下载**
   - Token会自动读取（从.env文件）
   - 点击 **"2024年全年"** 快速按钮
   - 点击 **"🚀 开始下载"**
   - 等待5-10分钟

4. **回到回测页面**
   - 下载完成后，重新运行回测即可

#### 方法2：直接回测（系统会自动在线获取）
- 如果没有本地数据，系统会自动使用Tushare API
- 速度较慢，但可以使用
- 不需要任何额外操作

## ❓ 常见问题解答

### Q1: 我看到学习实例04提到了gui_app/backtest，那是什么？

**A:** 那是旧的回测界面，已经不再使用。
- ❌ 旧系统：`gui_app/backtest/` (废弃)
- ✅ 新系统：`easyxt_backtest/` + 101因子平台（推荐）

**现在只需要使用101因子平台的策略回测功能即可。**

### Q2: easyxt_backtest是什么？我需要单独运行它吗？

**A:** 不需要！
- `easyxt_backtest` 是底层回测引擎
- 101因子平台会自动调用它
- 你只需要关注101因子平台的界面

**类比：**
- `easyxt_backtest` = 汽车引擎
- `101因子平台` = 汽车方向盘和仪表盘
- 你只需要开车（使用101平台），不需要关心引擎怎么工作

### Q3: 文档太多，我到底该看哪个？

**A:** 按优先级：
1. **这份文档**（快速上手）
2. `STRATEGY_BACKTEST_GUIDE.md`（详细功能说明）
3. `README.md`（101因子平台总览）

**不需要看：**
- ❌ `学习实例/04_策略回测.py`（旧教程）
- ❌ `markdown/04_策略回测教程.md`（旧教程）

### Q4: 回测失败怎么办？

**A:** 检查清单：

1. **数据问题**
   - 首次使用需要下载市值数据（见上文"下载方法"）
   - 或者选择"继续回测"让系统自动在线获取

2. **环境问题**
   - 确保安装了依赖：`pip install -r requirements.txt`
   - 确保Python版本 >= 3.7

3. **路径问题**
   - 确保 `easyxt_backtest` 在正确位置：
     ```
     C:\Users\Administrator\Desktop\miniqmt扩展\easyxt_backtest
     ```

### Q5: 我想修改策略参数怎么办？

**A:** 直接在界面上修改：
- 选股数量：调整持仓股票数
- 股票池大小：调整选股范围
- 初始资金：调整回测资金
- 时间范围：调整回测区间

**修改后点击"开始回测"即可，不需要重启！**

### Q6: 回测结果可以保存吗？

**A:**
- 当前版本：可以查看，可以导出JSON
- 后续版本：会增加导出CSV、图片等功能

## 🔧 技术架构说明（给进阶用户）

### 文件结构：

```
101因子分析平台/
├── main_app.py                      # 主应用入口
├── src/workflow/
│   └── strategy_backtest_page.py   # 策略回测页面（你看到的界面）
└── 调用 ↓

../easyxt_backtest/                  # 底层回测引擎
├── data_manager.py                  # 数据管理
├── engine.py                        # 回测引擎
└── strategies/
    └── small_cap_strategy.py       # 小市值策略
```

### 调用关系：

```python
# strategy_backtest_page.py 中的关键代码

# 1. 动态加载easyxt_backtest模块
import sys
sys.path.insert(0, str(project_root))

# 2. 导入回测引擎
from easyxt_backtest import DataManager, BacktestEngine
from easyxt_backtest.strategies import SmallCapStrategy

# 3. 创建并运行回测
data_manager = DataManager()
strategy = SmallCapStrategy(...)
engine = BacktestEngine(...)
results = engine.run_backtest(...)
```
