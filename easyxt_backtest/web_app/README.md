# 101因子平台 - Web应用使用指南

## 🚀 快速启动

### 方法1: 双击启动（推荐Windows用户）

双击项目根目录下的 `start_101_platform.bat` 文件即可启动。

### 方法2: 命令行启动

```bash
# 进入项目目录
cd C:\Users\Administrator\Desktop\miniqmt扩展

# 运行启动脚本
python easyxt_backtest\web_app\start_web_app.py
```

### 方法3: 直接使用Streamlit命令

```bash
streamlit run easyxt_backtest\web_app\streamlit_app.py
```

## 📱 访问界面

启动成功后，在浏览器中访问:

- **本地访问**: http://localhost:8501
- **局域网访问**: http://192.168.0.106:8501

## 🎯 功能说明

### 1. 📝 配置管理

- **加载示例策略**: 从侧边栏选择预设的示例策略
- **上传自定义配置**: 上传您自己的YAML配置文件
- **查看配置详情**: 查看策略的所有配置参数

### 2. 🔄 策略回测

- **设置回测参数**: 自定义开始和结束日期
- **运行回测**: 一键运行策略回测
- **查看回测结果**: 实时查看回测进度和结果

### 3. 📈 结果分析

- **性能指标**: 总收益率、年化收益率、最大回撤、夏普比率等
- **净值曲线**: 可视化展示策略净值走势
- **回撤曲线**: 查看历史回撤情况
- **交易记录**: 详细的交易历史记录

### 4. 💻 实盘代码生成

- **自动生成代码**: 根据YAML配置自动生成实盘交易代码
- **自定义输出**: 设置输出目录和QMT账户ID
- **查看代码**: 在线预览生成的代码
- **使用说明**: 详细的使用指南

## 📋 使用流程

### 完整工作流程

1. **配置策略**
   - 从侧边栏选择示例策略或上传自定义配置
   - 在"配置管理"页面查看和确认策略参数

2. **运行回测**
   - 切换到"策略回测"页面
   - 设置回测参数（开始/结束日期）
   - 点击"开始回测"按钮
   - 等待回测完成

3. **分析结果**
   - 切换到"结果分析"页面
   - 查看性能指标和图表
   - 分析策略表现

4. **生成实盘代码**
   - 切换到"实盘代码生成"页面
   - 设置输出目录和账户ID
   - 点击"生成实盘代码"
   - 查看生成的代码文件

- **交互式图表**: 使用Plotly实现交互式数据可视化

## 📊 示例策略

系统内置以下示例策略：

1. **simple_small_cap.yaml** - 简单小市值策略
   - 选择市值最小的10只股票
   - 等权重配置
   - 每月调仓

2. **multi_factor_strategy.yaml** - 多因子策略
   - 结合市盈率、市净率、市值等多个因子
   - 加权打分选股
   - 每月调仓

3. **alpha101_strategy.yaml** - Alpha101多因子策略
   - 使用Alpha101因子
   - 技术指标驱动
   - 每月调仓

## 📝 注意事项

1. **首次使用**: 请先安装所有依赖包
2. **端口占用**: 默认使用8501端口，如需修改请编辑启动脚本
3. **数据源**: 当前使用模拟数据，实际使用需配置真实数据源
4. **实盘交易**: 生成的实盘代码需谨慎测试后使用

## 🆘 常见问题

### Q: 启动失败怎么办？

A: 检查以下几点：
- 确保Python环境正确
- 安装所有依赖包
- 检查端口8501是否被占用

### Q: 回测很慢怎么办？

A: 当前使用模拟数据，真实数据回测速度取决于数据源性能。

### Q: 如何修改界面样式？

A: 编辑 `streamlit_app.py` 中的CSS样式部分。

### Q: 可以添加自定义因子吗？

A: 可以，在 `easyxt_backtest/factors/` 目录下添加自定义因子类。
