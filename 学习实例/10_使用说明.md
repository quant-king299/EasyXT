### 默认模式（推荐学习）


## 🔴 真实交易模式

### 方法1：命令行参数（推荐）

```bash
# 启用真实交易模式
python 学习实例/10_数据获取与交易结合案例.py --real-trading

# 指定股票代码
python 学习实例/10_数据获取与交易结合案例.py --real-trading --stock 600000

# 使用模拟数据
```

**启用真实交易时：**
1. 会显示风险警告
2. 需要输入 `YES` 确认才会启用
3. 确保您已安装QMT交易端
4. 确保您有交易账号权限

### 方法2：修改代码

找到第 `712-715` 行：

```python
strategy = TradingStrategy(
    use_real_data=True,      # 使用真实数据
    use_real_trading=False   # 👈 改成 True
)
```

## 🔧 故障排查

### 问题1：DataManager初始化失败

**错误信息：**
```
❌ DataManager初始化失败
```

**解决方法：**
- 检查是否安装了QMT/Tushare
- 确认数据源配置正确
- 查看详细错误信息

### 问题2：AdvancedTradeAPI未找到

**错误信息：**
```
⚠️ AdvancedTradeAPI模块未找到
```

**解决方法：**
- 这是正常的，会自动使用模拟交易
- 如需真实交易，需要完整安装easy_xt

### 问题3：无法启用真实交易

**可能原因：**
1. 没有安装QMT交易端
2. 没有交易账号权限
3. xtquant模块未正确导入

**解决方法：**
```bash
# 检查xtquant是否安装
pip list | grep xtquant

# 如果未安装，联系QMT获取安装包
```
