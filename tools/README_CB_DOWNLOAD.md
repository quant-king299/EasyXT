# 可转债数据下载工具

## 📁 工具说明

提供从QMT实时获取可转债列表并下载到本地的功能。

**核心优势**：
- ✅ 实时从QMT获取最新可转债列表
- ✅ 自动包含新上市的可转债
- ✅ 自动过滤无效代码（预留、退市等）
- ✅ 失败率接近0%

## 🚀 快速使用

### 方法1：批处理文件（推荐）

双击运行以下文件：

| 文件 | 用途 | 下载内容 | 适用场景 |
|------|------|----------|----------|
| `cb_test.bat` | 测试连接 | 4只知名可转债 | 首次使用，测试QMT连接 |
| `cb_quick.bat` | 一次性下载 | 2024年全年数据 | 首次建立2024年数据库 |
| `cb_download.bat` | 交互式下载 | 可选择不同时间段 | 灵活控制下载范围 |
| `cb_update_smart.bat` | 日常更新 | 最近1/3/6个月数据（可选） | 定期更新数据 |

### 方法2：GUI更新（最推荐）

**真正的日常更新推荐使用GUI**：

```
1. 打开GUI：python rungui.py
2. 切换到「数据管理」标签页
3. 点击「🔄 更新缺失数据」按钮
4. 系统智能检测并补充缺失数据
```

**GUI更新优势**：
- ✅ 智能检测每只股票最新日期
- ✅ 只下载真正缺失的部分
- ✅ 自动补充到今天
- ✅ 避免重复下载
- ✅ 最节省时间和资源

### 方法2：命令行

```bash
# 基本用法
python tools/download_all_convertible_bonds.py

# 指定日期范围
python tools/download_all_convertible_bonds.py --start-date 20240101 --end-date 20241231

# 测试模式
python tools/download_all_convertible_bonds.py --demo

# 强制重新下载
python tools/download_all_convertible_bonds.py --force
```

## 📊 下载流程

### 第1步：下载到QMT本地
```
运行批处理文件或命令
↓
从QMT实时获取可转债列表（约321只）
↓
下载历史数据到QMT本地
```

### 第2步：导入到DuckDB
```
打开GUI：python rungui.py
↓
切换到「数据管理」标签页
↓
点击「📥 下载可转债数据」按钮
↓
系统从QMT本地读取并导入到DuckDB
```

### 第3步：验证数据
```
在GUI中切换到「数据查看」标签页
↓
输入可转债代码（如 113061.SH）
↓
查看数据是否正常显示
```

## 📈 技术细节

### 获取方式

**优先级顺序**：
1. QMT板块实时获取（推荐，包含新上市）
2. 预定义列表（备用方案）

### 代码验证

自动过滤无效代码：
- 尚未发行的预留代码
- 已退市的可转债
- 暂停交易的代码
- 其他无法获取数据的代码

### 预期结果

- 获取数量：约321只（当前真实存在的可转债）
- 成功率：接近100%
- 失败数量：0-5只（临时停牌等）

## ⚠️ 注意事项

### 运行前检查
- [ ] QMT软件已启动
- [ ] 网络连接正常
- [ ] 有足够的磁盘空间

### 常见问题

**Q: 为什么获取到的可转债数量不是固定的？**
A: 因为是从QMT实时获取，数量会随着新上市和退市动态变化。

**Q: 可以直接用GUI下载吗？**
A: 可以！GUI会自动使用相同的逻辑获取列表，无需预先下载。

**Q: 下载失败怎么办？**
A:
1. 确认QMT是否正常启动
2. 检查网络连接
3. 查看错误日志
